# Movie List

*This is a movie list maker website built using Vue.js, a progressive JavaScript framework. The website allows users to create, edit, and delete movie lists.*

## Installation
Install [Node.js](https://nodejs.org/en/).

```bash
git clone https://github.com/Mr-R-b0t/scaling-octo-eureka.git

```
```bash
cd scalling-octo-eureka

```

```bash
npm i

```
If you have error use this command :

```bash
npm i --force
```

### How to run the code :

```bash
npm run serve
```

Then open your browser at this URL : [http://localhost:8080](http://localhost:8080).
